# Smart Facility Booking System

A web-based platform that digitizes facility booking and management for schools, colleges, and universities. Inspired by the group-and-code model used in apps like GradeChum, a school can create an organization on the platform and generate a unique join code, which is shared with students, faculty, and staff so they can register under that school and start booking shared facilities.

Once inside an organization, users can view real-time availability of facilities such as classrooms, auditoriums, sports facilities, and other shared venues, then submit booking requests for their preferred time slots. The system automatically detects scheduling conflicts and prevents double-bookings, suggesting alternative available time slots when a user's preferred slot is already taken.

Facility managers (e.g., school administrators) oversee all requests through a central dashboard, where they can approve or reject bookings, leave remarks, and use an admin override feature for urgent events (with mandatory remarks logged for transparency). Every action, who booked, who approved, and when, is recorded in a complete audit trail, replacing manual logbooks, whiteboards, and Excel sheets with a transparent, disputes-free process.

The platform also generates usage reports and key metrics for managers, including booking volume over time, pending/approved/rejected request counts, facility utilization rates, average approval time, and user satisfaction ratings, giving schools data-driven insight into how their facilities are actually used.

## Core Features

- Organization (school) creation with shareable join codes for members
- Real-time facility availability viewing
- Booking request submission with instant conflict detection
- Smart alternative time-slot suggestions
- Manager dashboard for approving/rejecting requests
- Admin override with mandatory remarks for urgent bookings
- Full audit trail of booking and approval actions
- Usage analytics and reporting dashboard
- Role-based access control (admin/manager vs. regular user)

## Target Users

Schools, colleges, and universities needing to manage shared facilities such as classrooms, auditoriums, and sports venues.
